通过异步接口批量提交抖音链接，拿到无水印视频地址、图集图片、标题作者与互动数据。规范 JSON、按次计费、稳定可靠——适合数据采集、内容备份与二次开发。

1快速开始

1

注册 / 登录

在首页右上角点「登录 / 注册」，完成滑块验证即可。

2

生成 API Key

进入 API 控制台点「生成密钥」，新密钥自带试用余额。

3

发起调用

在请求头带 X-API-Key 调用下方接口即可。

2鉴权

所有接口通过请求头 X-API-Key: 你的密钥 鉴权（也支持 ?key= 查询参数）。密钥在 API 控制台生成与管理。

X-API-Key: dy_xxxxxxxxxxxxxxxxxxxx

3接口

POST/api/v1/jobs
提交批量解析任务（异步）。传 links 数组或整段 text，立即返回 job_id。
# 请求体（二选一）
{ "links": ["https://v.douyin.com/xxxx/", "https://v.douyin.com/yyyy/"] }
{ "text":  "整段分享文案，自动提取其中所有链接" }

# 返回
{ "code": 0, "message": "accepted",
  "data": { "job_id": "job_xxx", "total": 2, "status": "pending",
            "price_cents": 1, "estimated_cost_cents": 2,
            "query_url": "/api/v1/jobs/job_xxx" } }
GET/api/v1/jobs/{job_id}
查询任务状态与结果。status 为 done 时 results 为逐条解析结果。
# 返回
{ "code": 0, "data": {
    "status": "done", "total": 2, "done": 2, "ok": 1, "cost_cents": 1,
    "results": [
      { "link": "...", "ok": true,  "data": { "title": "...", "author": "...",
          "video": { "url": "无水印视频直链", "width": 1080, "height": 1920 },
          "stats": { "digg": 12000, "comment": 340 } } },
      { "link": "...", "ok": false, "error": "链接已失效" }
    ] } }
GET/api/v1/balance
查询当前密钥余额与用量。
{ "code": 0, "data": { "balance_cents": 980, "spent_cents": 20, "calls": 20, "price_cents": 1 } }

4计费

每成功解析一条链接计费一次（默认 ¥0.01/条，可能调整）；解析失败不计费；余额不足会拒绝提交。余额可在控制台或 balance 接口查询，充值请联系作者。

5错误码

HTTP含义
400请求体错误 / 无合法链接
401缺少或无效的 API Key
402余额不足
404任务不存在或无权访问
需要更高并发 / 更低单价 / 私有部署？联系作者获取企业方案。
